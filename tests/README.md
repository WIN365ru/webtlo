# Автотесты

В каталоге `tests` лежит минимальный набор автотестов для проверки базового поведения кэш-слоя.

## Запуск

1. Установите зависимости приложения:
   ```bash
   cd src
   composer install
   ```

2. Запустите тесты из корня репозитория:
   ```bash
   php tests/run.php
   ```

Тесты используют временную директорию `tests/tmp` и создают в ней `config.ini` с отключённым Redis, чтобы проверять поведение `NullCache` и фабрики кэша без внешних сервисов.
